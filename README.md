# Bank Management System in C++

This project is a console-based Bank Management System developed using C++.

## Features
- Create bank account
- Deposit money
- Withdraw money
- Display account details
- File handling for persistent storage

## Technologies Used
- C++
- Object-Oriented Programming
- File Handling

## How to Run
1. Compile the program using a C++ compiler
2. Run the program
3. Use the menu options to perform banking operations
