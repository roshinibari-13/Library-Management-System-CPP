\# Library Management System in C++



This project is a console-based Library Management System developed using C++.



\## Features

\- Add books

\- Display book records

\- File handling for storing data

\- Menu-driven program



\## Technologies Used

\- C++

\- Object-Oriented Programming

\- File Handling



\## How to Run

1\. Compile the program using a C++ compiler.

2\. Run the program.

3\. Use the menu options to manage books.
